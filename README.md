## 📋 Sobre o Projeto

Pequenos estabelecimentos comerciais necessitam de sistemas simples e eficientes para controlar seus estoques, garantindo o registro correto de produtos, quantidades e valores. 

Este projeto é um sistema desenvolvido em **Python** (rodando em modo console) que visa solucionar essa necessidade. Através da utilização de estruturas de dados (listas e dicionários) e da modularização com funções, o sistema oferece uma solução organizada e de fácil manutenção para o apoio à gestão comercial.

---

## ✨ Funcionalidades

O programa apresenta um menu interativo com as seguintes opções:

* **Cadastrar Produto:** Permite a inserção de novos itens no estoque (Código, Nome, Preço e Quantidade).
* **Validação de Dados:** O sistema bloqueia o cadastro de códigos duplicados e impede a inserção de preços ou quantidades com valores negativos.
* **Calcular Total em Estoque:** Exibe a quantidade total de produtos armazenados, somando todas as unidades cadastradas.

---

## 🛠️ Estrutura Técnica

* **Linguagem:** Python
* **Armazenamento:** Lista de dicionários (dados em memória durante a execução).
* **Estrutura:** Código modularizado com o uso obrigatório de funções para cada operação do menu.
